# Project Setup

Follow these steps to set up the project locally for development.

## 1. Clone the Repository

```bash
git clone https://github.com/boniyeamincse/Anti-Diabetes-AI-App.git
cd Anti-Diabetes-AI-App/Anti_Diabetes/mobile_app
```

## 2. Get Dependencies

Run the following command to download all necessary packages:

```bash
flutter pub get
```

## 3. Verify Setup

Run `flutter doctor` again to ensure everything is ready for this specific project.

## 4. Open in IDE

Open the `Anti_Diabetes/mobile_app` folder in your preferred IDE (VS Code or Android Studio).
