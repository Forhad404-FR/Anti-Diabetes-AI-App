# Flutter Installation Guide

Follow these steps to set up Flutter on your machine.

## 1. Download Flutter SDK

Visit the official [Flutter Install](https://docs.flutter.dev/get-started/install) page and choose your operating system.

## 2. Extract and Add to Path

Extract the downloaded zip file and add the `flutter/bin` directory to your system's PATH environment variable.

## 3. Run Flutter Doctor

Open a terminal and run:

```bash
flutter doctor
```

This command checks your environment and displays a report of the status of your Flutter installation. Fix any issues reported (e.g., missing Android SDK, Xcode, etc.).

## 4. Install IDE Plugins

- **VS Code**: Install the "Flutter" extension from the Marketplace.
- **Android Studio**: Go to `Settings > Plugins` and install the "Flutter" plugin.
