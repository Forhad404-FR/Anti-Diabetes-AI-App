# Requirements

To develop and run the Anti-Diabetes AI App, ensure your system meets the following requirements.

## Development Environment

- **Operating System**: Windows, macOS, or Linux.
- **Flutter SDK**: Version `^3.11.0` (as specified in `pubspec.yaml`).
- **Dart SDK**: Automatically included with Flutter.
- **IDE**: 
  - [VS Code](https://code.visualstudio.com/) (with Flutter & Dart extensions) - **Recommended**
  - [Android Studio](https://developer.android.com/studio) (with Flutter & Dart plugins)

## Hardware Requirements

- **Memory**: Minimum 8GB RAM (16GB recommended for running emulators).
- **Disk Space**: At least 10GB for Flutter SDK, Android Studio, and Android SDK/Build Tools.

## Target Platforms

- **Android**: API Level 21 (Android 5.0) or higher.
- **iOS**: iOS 11.0 or higher.
- **Web**: Chrome, Firefox, Safari, or Edge.
