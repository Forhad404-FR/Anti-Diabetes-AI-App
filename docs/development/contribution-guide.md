# Contribution Guide

Thank you for your interest in contributing to the Anti-Diabetes AI App!

## How to Contribute

1. **Fork** the repository and clone it to your machine.
2. **Follow the Setup Guide** in `docs/setup/project-setup.md`.
3. **Create a Branch** using the naming convention in `docs/development/branching-strategy.md`.
4. **Write Code**: Ensure you follow the coding standards.
5. **Add Tests**: (Planned) add unit or widget tests for your new features.
6. **Submit a PR**: Detailed description of what you've done.

## Rules of Conduct

- Be respectful and collaborative.
- Provide clear documentation for any new features added.
- Ensure the app builds successfully on both Android and iOS before submitting.

## Reporting Issues

If you find a bug or have a suggestion, please use the GitHub Issues tracker and include:
- A clear title and description.
- Steps to reproduce the bug.
- A screenshot or screen recording if it's a UI issue.
