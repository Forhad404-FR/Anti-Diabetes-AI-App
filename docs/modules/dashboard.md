# Dashboard Module

The Dashboard is the central hub of the Anti-Diabetes AI App.

## Overview

The Dashboard provides users with a snapshot of their health data and quick access to tracking tools.

- **Location**: `lib/features/dashboard/`.
- **Key Widget**: `DashboardPage`.

## Key Features

- **Health Summary**: Visual representation of recent glucose, diet, and exercise status.
- **Quick Links**: Fast navigation to health trackers and the AI assistant.
- **Notifications**: Alerts and reminders related to diabetes management (planned).

## UI/UX Design

- Uses a card-based layout for readability.
- Integrated into the `BottomNavPage` as the default first tab.
